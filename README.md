# Intelligence QiKuo (颀阔智能)

## 项目简介
颀阔智能（Intelligence QiKuo）是一家懂资产运营的数据服务商，专注于为新能源、基础设施及各类持有型资产提供数字化运营与金融化服务。

## 技术栈
- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式库**: Tailwind CSS
- **动画库**: Framer Motion
- **图标库**: Lucide React

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 构建与部署
本项目使用 GitHub Actions 自动部署到 GitHub Pages。
每次推送到 `main` 分支时会自动触发构建。
